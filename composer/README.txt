Composer is required so that Log4php can be installed using it.

See the url: https://getcomposer.org

The download page is at: https://getcomposer.org/download

The software log4php is useful for logging and will be
installed via composer.json .

Here is the website for log4php:

http://logging.apache.org/log4php/

The install with composer is given by:

http://logging.apache.org/log4php/download.html

Here is the command to use:

php composer.phar install
